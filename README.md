# citycountry-4646

CityCountry is a fast-paced, multiplayer word game where players compete to name cities, countries, and other geographical terms starting with a random letter within a time limit. The MVP will be a web-based game with real-time leaderboards, simple matchmaking, and a clean, engaging interface to test core gameplay and user retention.

## Tech Stack
Next.js, Node.js with Socket.io, PostgreSQL, Tailwind CSS

## Setup
Run `docker-compose up` to start the infrastructure.
